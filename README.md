# 🏙️ Dubai Residential Property Market Analysis

An end-to-end Data Analytics project that explores Dubai's residential property market using Python, SQL, and Power BI. The project focuses on data cleaning, exploratory data analysis (EDA), feature engineering, and interactive dashboard development to uncover pricing trends, transaction distribution, and high-value investment areas.

---

## 📌 Project Overview

Dubai has one of the fastest-growing real estate markets in the world. Understanding pricing patterns, transaction volume, and property distribution can help investors, developers, and buyers make informed decisions.

This project analyzes more than **18,000 residential property transactions** to identify:

- Market overview
- Price distribution
- Most expensive locations
- Property type performance
- Monthly transaction trends

---

## 🎯 Objectives

- Clean and prepare raw property transaction data
- Perform Exploratory Data Analysis (EDA)
- Create summary datasets for reporting
- Develop an interactive Power BI dashboard
- Generate actionable business insights

# 📈 Key Insights

### 🏘️ Property Market

- Flats dominate residential transactions.
- Luxury properties contribute significantly to total transaction value.
- Mid-range properties account for the largest transaction volume.

### 📍 Area Performance

- Business Bay and Palm Jumeirah consistently rank among the highest-value locations.
- Premium waterfront districts command substantially higher prices per square meter.

### 💰 Pricing

- Large price differences exist between districts.
- Premium locations can be several times more expensive than emerging residential areas.
